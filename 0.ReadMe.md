# CDS95Util
『대항해시대 III』 Windows 95용 유틸리티

## 개요
『대항해시대 III』 Windows 95판(이하 CDS95)에 다양한 기능을 추가하거나 기존 동작을 변경하는 유틸리티 모음입니다.
플러그인 방식으로 구성되어 있어 필요한 기능만 선택하여 사용할 수 있습니다.

## 지원 운영체제
Windows 2000 이상 (32비트 및 64비트 모두 지원)

## 사용 방법
Bin 폴더 안의 모든 파일을 CDS95의 설치 폴더에 복사하십시오.
CDS95.exe와 같은 폴더에 ddraw.dll이 존재하면 정상적으로 설치된 것입니다.
단, 이미 ddraw.dll 파일이 있는 경우에는 본 유틸리티를 사용할 수 없습니다.

또한 일부 플러그인은 설정 파일이나 저장 데이터를 기록하므로 CDS95 설치 폴더에 대한 쓰기 권한이 필요합니다.
만약 게임을 Program Files 폴더에 설치한 경우에는 다음 중 한 가지 방법을 사용하십시오.
- 다른 경로에 다시 설치하거나
- 설치 폴더의 접근 권한을 수정하여 쓰기 권한을 부여하십시오.

## 재배포·개조·전재에 관한 조건
5ch.net의 『대항해시대』 관련 스레드에 한해서는 자유롭게 전재 및 재배포하셔도 됩니다.
또한 향후 업로더에서 파일이 삭제되는 경우에는 사전 허가 없이 다시 업로드하셔도 됩니다.
본 유틸리티의 출처와 수정한 부분을 명확히 표시하는 조건이라면, 수정한 버전을 재배포하는 것도 허용합니다.

다만 직접적이든 간접적이든 금전이 오가는 일에는 관여하고 싶지 않으므로,
유료 프로그램 관련 사이트 등에 게시하거나 재배포하는 것은 삼가 주시기 바랍니다.

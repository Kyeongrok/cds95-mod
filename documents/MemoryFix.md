# MemoryFix
CDS95.exe 메모리 버그 대책 플러그인

## 개요
CDS95.exe의 메모리 관련 버그를 흡수하여 CTD(강제 종료)가 발생하기 어렵게 만드는 플러그인입니다.
- CDS95의 GlobalAlloc 계열 API를 후킹하여 동작을 변경합니다.
- Windows 10 등에서도 Windows 95 호환 모드로 설정하지 않아도 동작하도록 합니다.
- 게임 내 계산기를 사용하려고 할 때 발생하는 CTD를 방지합니다.

## 대응 버전
CDS95.exe 버전에 대한 의존성은 없습니다.
모든 버전에서 동작할 것으로 예상되지만, Ver.1.1.0.0과 Ver.1.2.7.0 이외는 미확인입니다.
버전에 따라서는 해당 버그가 존재하지 않을 수도 있기 때문에 불필요한 동작이 늘어날 수도 있지만, 동작에는 지장이 없습니다.

## 내부 동작
다음은 Ver.1.2.7.0 기준 검증 결과입니다.

CDS95.exe는 일부 구간에서 GlobalAlloc API를 사용하여 메모리를 할당하고 여러 작업을 수행하지만, 할당된 메모리 주소보다 앞쪽 메모리를 덮어쓰는 경우가 있습니다. 이는 일종의 버퍼 오버런 버그입니다.
현재까지는 1바이트 앞을 덮어쓰는 패턴만 확인되었습니다. 이로 인해 메모리가 손상되어 CTD가 발생할 수 있습니다.

본 플러그인은 CDS95.exe가 GlobalAlloc으로 할당하는 메모리의 앞뒤에 각각 256바이트의 완충 영역을 두어, 중요한 메모리가 파괴되는 것을 방지합니다.
이로 인해 메모리 사용량이 증가하고 할당 시간도 늘어나지만, CTD가 발생하는 것보다는 낫다고 판단했습니다.

또한 Win32 API 규격 범위 내이긴 하지만, GMEM_MOVEABLE을 사용하지 않고 GlobalAlloc으로 할당한 메모리에 대해 왜인지 GlobalLock / GlobalUnlock을 호출하는 동작도 확인되었습니다.
Microsoft 문서에 따르면 오류는 아니지만 불필요한 동작이며, 사용 방식을 제대로 이해하지 못한 것으로 보입니다……

본 플러그인은 CDS95.exe에서의 직접 호출에만 반응하며, 다른 DLL 등의 호출에는 개입하지 않아 Windows 본래 동작이 유지되도록 설계되어 있습니다.
